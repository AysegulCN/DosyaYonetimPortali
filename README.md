Dosya Yönetim Portalı (CoreDrive)
Bulut tabanlı, modern ve kullanıcı dostu bir dosya yönetim sistemi. Kullanıcıların dosyalarını güvenle saklayabileceği, paylaşabileceği ve yönetebileceği bir platformdur.

🚀 Temel Özellikler:

Klasör Hiyerarşisi: İç içe klasör oluşturma ve düzenleme.

Dosya Yönetimi: Sürükle-bırak yükleme, önizleme ve indirme desteği.

Paylaşım Sistemi: Dosyaları diğer kullanıcılarla paylaşma ve bağlantı kopyalama.

Gelişmiş Admin Paneli: Sistem logları, depolama analizi ve kullanıcı yönetimi.

Modern UI: Google Drive esintili, responsive (mobil uyumlu) arayüz.

🛠 Teknolojiler:

Backend: ASP.NET Core MVC & Web API (C#)

Database: MS SQL Server (Entity Framework Core)

Frontend: Bootstrap 5, FontAwesome, JavaScript

Raporlama: iText7 (PDF Raporlama)

## 📂 Proje Yapısı
*   **DosyaYonetimPortali.API:** Veritabanı modelleri ve iş mantığının yönetildiği arka uç.
*   **DosyaYonetimPortali.MVC:** Kullanıcıların dosyalarını yönettiği ön yüz (arayüz).
*   **wwwroot/uploads:** Kullanıcıların yüklediği fiziksel dosyaların saklandığı güvenli alan.
